# power-user-cultivation

**Turn users into advocates and contributors** — identifying, nurturing, and empowering your developer champions.

## What It Does

Helps you identify your most engaged developers and create programs that turn them into advocates, content creators, and contributors. No forced evangelism — genuine value exchange for developers who love your product.

## When to Use

- Identifying potential advocates in your user base
- Building a champion or ambassador program
- Creating contributor experience for open source
- Launching a referral program
- Starting user-generated content initiatives
- When you say "power users," "developer advocates," "champion program," or "UGC"

## Quick Start

```
/power-user-cultivation

Help me identify potential advocates from our user base and design
a champion program. We have about 5,000 active developers using our API.
```

## The Power User Spectrum

| Level | Behavior | Your Response |
|-------|----------|---------------|
| Active User | Uses product regularly | Keep shipping |
| Engaged User | Participates in community | Respond quickly |
| Advocate | Recommends unprompted | Early access, direct line |
| Champion | Creates content, helps others | Formal program |
| Contributor | Contributes code/docs | Contributor program |

## Key Principles

1. **Don't push everyone up the spectrum — meet them where they are**
2. **No content quotas — advocates create because they want to**
3. **Credits > cash for referrals (less MLM feeling)**
4. **Recognition must be specific, public, and timely**
5. **Start small (5-10 champions) and grow**

## Identifying Advocates

Look for signals:
- Top 10% by usage
- Answers community questions
- Created content about you
- Files thoughtful bug reports
- Positive social mentions
